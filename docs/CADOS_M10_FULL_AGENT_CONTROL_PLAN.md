The keystone generalizes this to any op_id behind a registry allow-list + write-mode gate.

**Safety invariants preserved in every phase** (no regression): original DWG READ-ONLY (staged copy only, sha verified); no-fake-success; raw command never agent-exposed; `write_original` never permitted from the agent surface; blocked/unknown ops refused (not executed).
