## 구조 판정 — 삭제형 교란의 정보 한계 (B4 원리의 일반화)

CE-A류(교란이 자기 흔적을 완전히 지워 결과 장면이 어떤 정직 장면과 관측-동치가 되는 삭제형)는
B4와 동일한 수학이다: 무상태 추정기는 "원래 깨끗했던 장면"과 "오염 후 깨끗해 보이는 장면"을
구별할 수 없고, 후자의 상승을 금지하면 전자의 정당한 신뢰도도 금지된다. 열거식 사후 면제는
두더지잡기다. **L1e의 봉인은 관측-동치 증인 기준으로 원리화한다**:

> 상승 허용 ⇔ 교란 후 장면과 관측-동치이며 그 상태를 도시에 의미론상 정당하게 받는 **정직 증인
> 장면**을 제시할 수 있다. 증인 불가능한 상승(의심 신호 잔존: CE-B2·CE-D류) = 위반.

이는 밴드 완화가 아니라 2차 함대에서 B4에 이미 수용한 원리의 봉인 전 일반화다.

## L1e 수리 계약 골격

1. **의미론 수리 (증인-불가 상승 2건 + 부수 2건)**: ① CE-B2 — 잔존 ratio-outlier 등 가시적 의심
   신호는 reference 신뢰도에 지속 감쇠로 반영(재타입·display 삭제로 소거 불가한 방식) ② CE-D —
   eligible 공집합/단독 handle 시 NONE 유지(단독 잔존 증거가 분자가 되는 경로 제거, 최소 독립
   지지 요건) ③ D1 — 부분집합 판정을 record 수가 아니라 handle-집합 기준으로 ④ D3 — 이진 절벽을
   연속 감쇠로(τ 경계 미세 이동에 강건).
2. **봉인 외부 증인**: 2단계 실행 — Phase A: 워커가 prereg+봉인 워크북만 산출·정지 → **오케스트
   레이터가 git 커밋**(repo 이력=append-only 증인, 커밋 SHA가 선행 증명) → Phase B: 워커가 커밋된
   봉인 SHA를 인용하며 추정기 구현·실행. 로컬 시각 자기 보고 폐기.
3. **속성 시험 pool 다양화**: 0-절벽 시작 상태·ratio outlier·혼합 공간·handle 충돌·근접-τ 산포를
   pool에 의무 포함 + 상승 발생 시 증인 분류(증인 제시=한계 기록 / 증인 불가=위반) 자동화.
   항진식 술어 수리.
4. 함대 3차 프로브(CE-A~F·D1~D4·seat4 1,548-장면 탐색) 회귀 편입.
